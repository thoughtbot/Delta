ActionType

public protocol ActionType

This protocol is used when you want to make modifications to the store’s state. All changes to the store go through this type.

Sample Action:

struct UpdateIdAction: ActionType {
    let id: Int

    func reduce(state: AppState) -> AppState {
        state.id.value = id
        return state
    }
}

store.dispatch(UpdateIdAction(id: 1))
  • reduce(_:)

    This method is called when this action is dispatched. It’s purpose is to make modifications to the state and return a new version of it.

    Note

    This is the only place that changes to the state are permitted.
